Without_TTS.py:
* TTS ย่อมาจาก Text to speech

---
With_TTS.py:
* TTS ย่อมาจาก Text to speech
อันนี้ต้้องโหลดเพิ่ม
+ มีเสียงพูดให้
ต้องโหลด gTTS (Google Text To Speech)

โดย วิธีการโหลดคือ:
* เปิด Terminal ของคุณ แล้วพิมพ์คำสั่งว่า
  * *pip install gTTS*


# How to use:
 * Mobile:
